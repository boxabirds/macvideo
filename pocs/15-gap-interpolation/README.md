# POC 15 — Interpolated shots for instrumental gaps

**Goal:** Real songs have gaps between sung lines (breaths, instrumental bridges, solos). POC 13's gap-inclusive stretching hides small gaps but goes visually stale on longer ones. This POC inserts a dedicated keyframe + LTX clip for each significant gap, with the prompt generated by asking the LLM to interpolate between the neighbouring lyric shots.

## Pass criteria

- [ ] Gaps between lyric lines detected using a threshold (default 1.5 s)
- [ ] Above threshold: new interpolated shot created; below: small gap absorbed into prior clip
- [ ] Interpolated prompts reference the neighbouring lyrical shots' prompts, honour the world brief, and connect visually to both sides
- [ ] Identity chain continues across interpolated shots
- [ ] Final video has N + K shots for N lines with K significant gaps
- [ ] Continuous audio holds through the whole sequence
- [ ] Hard cuts land on content boundaries (line start or gap start)

## Architecture

Forks POC 13's architecture with two additions:

1. **Gap detection** in `prep.py` — after resolving line timings, scan pairs `(line[i].end, line[i+1].start)`. For each gap > threshold, create a gap shot entry between them.
2. **Interpolation prompt** — for each gap, LLM is given the prior and next image prompts plus the world brief, and asked to write a prompt for an instrumental moment fitting between them.

Everything else identical to POC 13 v2: Pass A world brief, identity chain (Gemini passes all prior keyframes as refs), affirmative-only prompts, continuous audio, libx264 concat.

## Test scope (same verse-1 segment as POC 13 v2)

| # | Shot type | Timing | Duration |
|---|---|---|---|
| 1 | line: "He's arrived again" | 15.86–17.28 | 1.42 s |
| 2 | gap (interpolated) | 17.28–19.04 | 1.76 s |
| 3 | line: "Whispering in my ear" | 19.04–21.10 | 2.06 s |
| 4 | gap (interpolated) | 21.10–23.18 | 2.08 s |
| 5 | line: "That's my little blackbird" | 23.18–24.98 | 1.80 s |

Threshold set to 1.5 s for this test so both gaps interpolate.

## How to run

```bash
uv run python pocs/15-gap-interpolation/scripts/prep.py
bash   pocs/15-gap-interpolation/scripts/run.sh
```

## Output

- `character_brief.json`
- `shots.json` (all 5 shots with timings, types, prompts)
- `keyframe_01.png` ... `keyframe_05.png` (lyric and gap keyframes, identity-chained)
- `clip_01.mp4` ... `clip_05.mp4`
- `audio_full.wav`
- `final.mp4`
- `prompts.json` (every prompt used)
